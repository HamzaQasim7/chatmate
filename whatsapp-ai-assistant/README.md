## ğŸ› ï¸ Installation

### For Development
1. Clone the repository
   ```bash
   git clone https://github.com/yourusername/reple.git
   ```
2. Install dependencies
   ```bash
   npm install
   ```
3. Start the development server (supports Hot Module Replacement)
   ```bash
   npm run dev
   # OR for specific browsers:
   npm run dev:firefox
   ```
4. Load the extension in Chrome/Edge/Brave:
   - Go to `chrome://extensions`
   - Enable "Developer Mode"
   - Click "Load Unpacked"
   - Select the `.output/chrome-mv3` folder

### For Users
1. **[Install from Chrome Web Store](#)** (Coming Soon)
2. Click the extension icon and enter your **OpenAI API Key**.
3. Open WhatsApp Web, LinkedIn, or Slack.
4. The Reple sidebar will appear automatically when you open a chat.
